|------|-----|

| `villian.png` | Boss (villain) ship |
| `commet.png` | Comet hazard |
| `sat.png` | Satellite hazard |
| `alien.png` | Alien hazard |

Legacy SVG placeholders (ship1.svg, etc.) are no longer used; the game uses these PNGs.
